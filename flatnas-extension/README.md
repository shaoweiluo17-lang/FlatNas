# FlatNas 浏览器扩展

轻量级、高度可定制的个人导航页与仪表盘系统 - 浏览器扩展版

## 功能特性

- ✅ 快速访问 FlatNas 服务
- ✅ 自定义新标签页
- ✅ 支持多种部署环境（本地、局域网、互联网）
- ✅ 服务状态监控
- ✅ 可自定义服务地址
- ✅ 自动在新标签页中打开 FlatNas

## 安装方法

### Chrome / Edge / Brave (推荐)

1. 下载或解压 `flatnas-extension` 文件夹
2. 打开浏览器，访问 `chrome://extensions/` 或 `edge://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `flatnas-extension` 文件夹
6. 扩展安装完成！

### Firefox

1. 下载或解压 `flatnas-extension` 文件夹
2. 打开浏览器，访问 `about:debugging#/runtime/this-firefox`
3. 点击"临时载入附加组件"
4. 选择 `flatnas-extension` 文件夹中的 `manifest.json` 文件
5. 扩展安装完成！

## 使用方法

### 1. 配置服务地址

1. 点击浏览器工具栏中的 FlatNas 图标
2. 点击"设置"按钮
3. 在"FlatNas 服务地址"中输入您的服务地址
   - 本地开发：`http://localhost:3000`
   - 局域网访问：`http://192.168.x.x:23000`
   - 互联网访问：`https://your-domain.com`
4. 点击"保存设置"

### 2. 使用扩展

**快速访问**
- 点击浏览器工具栏中的 FlatNas 图标
- 点击"打开 FlatNas"按钮

**新标签页**
- 开启"自动在新标签页中打开 FlatNas"选项
- 每次打开新标签页时，会自动跳转到 FlatNas

**设置自定义新标签页**
- 如果您希望使用 FlatNas 作为新标签页：
  1. 点击"设置"
  2. 在浏览器地址栏中输入 `chrome://extensions/`
  3. 找到 FlatNas 扩展
  4. 点击"详细信息"
  5. 在"扩展程序选项"中，点击"在以下位置上允许访问文件"（如果需要）
  6. 在浏览器设置中，将新标签页设置为 FlatNas 提供的页面

## 技术细节

### 支持的服务地址

扩展支持以下类型的服务地址：
- `http://localhost:*` - 本地开发环境
- `http://192.168.*` - 局域网 C 类地址
- `http://10.*` - 局域网 A 类地址
- `http://172.*` - 局域网 B 类地址
- `https://*` - HTTPS 安全连接

### 权限说明

扩展需要以下权限：
- `storage` - 保存用户设置
- `alarms` - 定期检查服务状态
- `tabs` - 管理标签页
- `webNavigation` - 监听页面导航

### 隐私说明

- 扩展不会收集任何用户数据
- 所有设置都保存在本地
- 不会向第三方发送任何信息

## 故障排除

### 问题：无法连接到 FlatNas 服务

**解决方案：**
1. 检查服务地址是否正确
2. 确认 FlatNas 服务正在运行
3. 检查网络连接
4. 如果使用 HTTPS，确保证书有效

### 问题：新标签页不生效

**解决方案：**
1. 确认已启用"自动在新标签页中打开 FlatNas"选项
2. 检查浏览器的新标签页设置
3. 重新加载扩展

### 问题：扩展图标不显示

**解决方案：**
1. 确认扩展已正确安装
2. 刷新浏览器页面
3. 检查浏览器是否禁用了该扩展
